# Embedded Modules

## ArduinoIDE Setup + Libraries

Using the ArduinoIDE, you should set this folder (`S25-EmbeddedSecuritySystem/embedded`) to be **Sketchbook Location**.  You can change this in the Preferences window.

If you do the above, when you add libraries, the code will be added to the `libraries` sub-folder and should be committed along with your module code.
